# Heart-disease-predictor
The AI Heart Disease Predictor is a web-based healthcare application that uses machine learning to assess a person's risk of developing heart disease. Users enter medical information such as age, blood pressure, cholesterol levels, heart rate, and other health indicators.
# AI Heart Disease Predictor

An AI-powered web application that predicts the risk of heart disease using machine learning algorithms.

## Features
- Heart disease risk prediction
- Patient data analysis
- Interactive dashboard
- Data visualization
- Prediction history tracking

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS
- SQLite

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python trainn.py

3. Start the application:
   python appp.py

4. Open:
   http://127.0.0.1:5000
